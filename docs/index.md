## 目录分工

- `design-docs/`
  长期设计知识、领域上下文、设计信念和决策。
- `product-specs/`
  产品规格、业务目标、范围和验收标准。
- `exec-plans/`
  进行中的计划、已归档计划和技术债跟踪。
- `references/`
  可重复查阅的外部资料摘要。
- `generated/`
  机器生成的事实快照。
